# LaTeX 수학 기호 가이드

자주 사용되는 LaTeX 수학 기호와 해당 코드를 정리한 문서입니다.

## 화살표 기호

| 기호 이름 | LaTeX 코드 | 렌더링 예시 |
|:---:|:---:|:---:|
| 왼쪽 화살표 | `\leftarrow` | $\leftarrow$ |
| 오른쪽 화살표 | `\rightarrow` | $\rightarrow$ |
| 양방향 화살표 | `\leftrightarrow` | $\leftrightarrow$ |

## 사칙연산 기호

| 기호 이름 | LaTeX 코드 | 렌더링 예시 |
|:---:|:---:|:---:|
| 곱셈 | `\times` | $\times$ |
| 나눗셈 | `\div` | $\div$ |

## 비교 연산자 (부등호)

| 기호 이름 | LaTeX 코드 | 렌더링 예시 |
|:---:|:---:|:---:|
| 작거나 같음 | `\leq` | $\leq$ |
| 크거나 같음 | `\geq` | $\geq$ |
| 같지 않음 | `\neq` | $\neq$ |

## 기타 유용한 기호

| 기호 이름 | LaTeX 코드 | 렌더링 예시 |
|:---:|:---:|:---:|
| 무한대 | `\infty` | $\infty$ |
| 근호 (루트) | `\sqrt{x}` | $\sqrt{x}$ |
| 합계 (시그마) | `\sum` | $\sum$ |
