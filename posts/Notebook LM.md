NotebookLM을 제대로 활용하려면, 먼저 이 도구가 어디서 어떻게 만들어졌는지 이해할 필요가 있습니다. 바로 구글의 AI 실험실, **Google Labs**입니다.

### Google Labs란?

Google Labs는 구글이 개발 중인 AI 실험(Experiments)을 한곳에 모아 공개하고, 사용자가 직접 체험하며 피드백을 남길 수 있게 하는 **‘AI 실험 허브’**입니다.

👉 <a href="https://labs.google/experiments" target="_blank">**[Google Labs 실험실 구경가기]**</a>

#### 🧪 AI 실험의 쇼케이스(체험 공간)
구글 전반에서 진행되는 최신 AI 실험/도구를 **‘먼저’** 발견하고 사용해볼 수 있습니다.
여기서 말하는 ‘먼저’는 정식 제품이 아니라 실험 단계 기능을 뜻하며, 실험은 수시로 바뀌거나 종료될 수 있습니다.

#### 🗣️ 피드백 기반 개선
사용자는 실험을 쓰면서 피드백을 제품 팀에 직접 전달할 수 있고, 구글은 이를 바탕으로 기술을 제품으로 확장(스케일)하는 과정을 설명합니다.
예: **SGE(Search Generative Experience)**는 Search Labs에서 “실험”으로 먼저 제공하며 개선에 사용자 피드백을 반영한다고 명시되어 있습니다.

#### 🔎 Search Labs와의 관계(실제로 자주 헷갈리는 포인트)
Google Labs 안에는 **Search Labs(검색 관련 실험 공간)**도 포함되어, 검색 실험과 도구들을 따로 모아 제공합니다.
NotebookLM도 Search Labs의 실험 항목으로 안내됩니다.


## NotebookLM이란?

- **Source-grounding**을 통해 사용자가 업로드한 소스만을 근거로 분석하고 심층 이해하여 처리하는 **AI 연구 비서**입니다.
- 인터넷 전체의 불확실한 정보 대신, **내가 검증한 자료만을 기반**으로 분석하고 응답합니다.

> 📌 **핵심 차별점: Source-grounding**
> AI가 "감으로 말하지 않고", 사용자가 제공한 **실제 자료(출처)**만을 근거로 답합니다. 이를 통해 **환각(Hallucination)**을 최소화하고 높은 신뢰성을 보장합니다.


### 📂 지원하는 소스 유형
NotebookLM은 다양한 형태의 정보를 한곳에 모아 처리할 수 있습니다.

- **📄 문서**: Google Docs, PDF, 텍스트 파일 (.txt, .md)
- **📊 슬라이드**: Google Slides
- **🌐 웹사이트**: 웹페이지 URL, 복사된 텍스트
- **🎧 멀티미디어**: YouTube 영상 (자막/스크립트 자동 분석), 오디오 파일 (.mp3, .wav)


### AI 인터랙션 작업

- **요약** 및 **질의응답**
- **스터디 가이드** / **FAQ** 생성
- **타임라인** · **브리핑** 작성
- **브레인스토밍** / **초안 작성**


### 출처 기반 응답

- 모든 답변에 **명확한 인용 제시**
- 신뢰성 및 추적성 강화


### 사용 시 포인트

| 포인트 | 설명 |
|--------|------|
| 📌 제한된 지식 범위 | 업로드 자료 외 지식 접근 불가 → 필요한 정보는 직접 추가 |
| ✅ 인용 검증 필수 | 답변의 인용 소스로 반드시 검증 |
| 📂 노트북 분리 | 주제별 노트북 분리 시 정확도 향상 |


### ⚡️ 핵심 기능

1.  **초고속 요약 (Summarize)**
    - 수백 페이지의 논문, 보고서, 회의록을 단 몇 초 만에 핵심만 요약합니다.
2.  **심층 질의응답 (Q&A)**
    - "이 문서에서 A와 B의 차이점이 뭐야?"처럼 자료를 기반으로 복잡한 질문에 답합니다.
3.  **콘텐츠 생성 (Generate)**
    - 자료를 바탕으로 블로그 초안, 스터디 가이드, 타임라인, FAQ 등을 자동 생성합니다.
4.  **오디오 개요 (Audio Overview)**
    - 딱딱한 텍스트 정보를 두 AI 호스트가 대화하는 **팟캐스트 형태**로 변환하여 들려줍니다.

## 참고 자료
### Grabbit
**Grabbit**은 크롬 브라우저 확장 프로그램으로, **웹페이지에서 여러 링크를 드래그로 선택해 일괄 처리**할 수 있는 도구입니다.

#### 주요 기능
- **드래그로 링크 선택**: 영역 내 링크 자동 인식 및 선택 (Smart Select, 중복 제거 지원)
- **다양한 액션 수행**: 선택된 링크를 새 탭/창으로 열기, 클립보드 복사, 북마크 생성 등
- **탭 관리**: 현재 탭 전체 URL 복사 또는 클립보드 링크 일괄 열기

<br>

#### URL 복사 Action 설정

**1. Grabbit 설치**
  - <a href="https://chromewebstore.google.com/detail/grabbit/madmdgpjgagdmmmiddpiggdnpgjglcdk?hl=ko&utm_source=ext_sidebar" target="_blank">구글 브라우저 확장프로그램: Grabbit</a>

**2. 설정 순서**
  - 브라우저 상단바에 **Grabbit 확장프로그램 고정**
  - 아이콘 클릭 후 **Options(설정)** 진입
  - 설정 화면 하단 **ADD New ACTION** 클릭
  - 아래와 같이 세부 옵션 설정:
    - **Mouse button**: `Right Click` (권장)
    - **Modifier key**: `Shift` (권장)
    - **Box color**: `원하는 색상 선택`
    - **Action**: `Copy URLs to cliboard`
    - **Advanced Options**:
      - `Smart select (heading filter): 활성` (불필요한 텍스트 제외)
      - `Avoid duplicates: 활성` (중복 링크 제거)
